Who am I

Hey, I'm a 22-year-old game developer/designer from North Carolina. I have a passion for all things computer-related, which is why I went into computer science. Ever since I began college in 2017, I've been working hard to learn new skills and refine the ones I have, from software programming in java to team management and planning. I graduated in 2021, with Chronologic being my final senior project. During that last senior semester, I was able to handle:

  • Making Chronologic in 3 out of the 4.5~ months

  • Working two part-time jobs

  • Managing a team for a java networking project

  • And completing my other classes

After graduating, I began working full-time at my family's farm/grain mill. During which I also began working on Eternal Soul in my free time.

Skills

Unity Engine Development

Used to create Chronologic, a 3D puzzle platformer inspired by Portal 2. It tests the player's understanding of physics and how things change when time stops. I have experience with character controls, lightmaps, light probes, object holding, physic interactions, level design, time, etc. If you want a more in-depth breakdown with visual examples, check out the Chronologic page.


Source code can be viewed here (github.com)


Gamemaker Studio 2 Development

Used to create Eternal Soul, a 2D Metroidvania inspired by Koji Igarashi's Castlevania games. I have experience creating character state machines, enemy logic, UI elements, interactive menus, reflections, inventory systems, combat interactions, equipable items, shattering objects, temporary effects, etc. If you want a more in-depth breakdown with visual examples, then check out the page for Eternal Soul.


The source code is not public. This is a private project and will potentially become a fully released game. I am dedicated to seeing it through to the end. If you're interested in knowing how specific parts work, feel free to ask. I'm willing to explain and share code snippets! (Within reason)


I also used GMS2 to create two other games during college. The Case Files, and Arctic Assault. These were both early learning experiences and do not remotely reflect my current skills. However, their source code is linked via Github by clicking the text above for transparency.


Aseprite Pixel Art

Used to create all of the art in Eternal Soul. Creating pixel art is not my primary skill (programming), but I've worked hard over the last year to refine it. I don't have a repository of every sprite I've created, but I can share examples if needed. I have experience creating items, characters, animations, enemies, UI, tilesets, particles, and more.


Story, Lore, and Worldbuilding

I love writing stories and have already finished a portion of Eternal Soul's lore. I take inspiration from music, the environment, and existing media. I try to create complex stories that tie its various games together and games where everything in it is explainable.


I've used Word, Google Docs, and Pages for my writing. They're all fine but have a bias towards Docs for the online convenience. I also use Campfire Pro to track character info, backstories, relationships, abilities, limitations, key items, locations, timelines, etc.
